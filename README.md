Program ini berfungsi untuk menghitung nilai rata-rata ujian siswa.
Cara kerjanya:
Pengguna memasukkan jumlah siswa.
Program kemudian meminta nilai masing-masing siswa satu per satu dengan perulangan for.
Semua nilai dijumlahkan dan hasilnya dibagi dengan jumlah siswa untuk mendapatkan rata-rata.
Hasil rata-rata ditampilkan ke layar.
